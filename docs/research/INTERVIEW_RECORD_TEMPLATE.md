# Interview Record Template

Status: Template
Last updated: 2026-04-21
Purpose: Fill this immediately after each interview so raw evidence does not stay scattered across memory or chat notes.

## How to Use

- Copy this template once per participant.
- Fill it immediately after the interview while details are still fresh.
- Prefer concrete episodes and direct quotes over summaries like "they want feedback."
- If the participant said something surprising, keep the original wording.
